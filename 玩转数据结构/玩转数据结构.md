### 二分搜索树

二叉树

![](img/13.jpg)



多叉数



二叉树具有唯一的根节点

```java
class Node{
  E el;
  Node left;//左孩子
  Node right;//右孩子
}
```

二叉树每个节点最多有两个孩子

一个孩子都没有的节点 通常称为叶子节点

二叉树每个节点最多有一个父亲



二叉树具有天然递归结构

​	每个节点的左子树也是二叉树

​	每个节点的右子树也是二叉树

二叉树不一定是满的

![](img/14.jpg)

二分搜索树：

​	二分搜索树也是二叉树

​	二分搜索树的每一个节点的值：

​		大于其左子树节点的值

​		小于其右子树节点的值

​	每一棵子树也是二分搜索树

![](img/15.jpg)
